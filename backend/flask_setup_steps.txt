
1. Instale o Python (se ainda não estiver instalado)
   - Baixe e instale a versão mais recente do Python 3.x em https://www.python.org/downloads/.
   - Certifique-se de marcar a opção “Add Python to PATH” durante a instalação no Windows ou instalá-lo via gerenciador de pacotes no Linux/macOS, para que o comando `python` esteja disponível no terminal.

2. Abra um terminal ou prompt de comando
   - No Windows, você pode usar `cmd` ou `PowerShell`.
   - No macOS ou Linux, abra o Terminal.

3. Navegue até a pasta do projeto
   - Use `cd` para entrar na pasta onde o projeto foi descompactado (por exemplo, `cd caminho/para/consulta-escola-sme/backend`).

4. Crie um ambiente virtual (recomendado)
   - Crie um ambiente virtual chamado `venv` dentro da pasta `backend`:
     ```bash
     python -m venv venv
     ```
   - Ative o ambiente virtual:
     - No Windows:
       ```bash
       venv\Scripts\activate
       ```
     - No macOS/Linux:
       ```bash
       source venv/bin/activate
       ```

5. Instale as dependências do projeto
   - Ainda no diretório `backend` (e com o ambiente virtual ativo), execute:
     ```bash
     pip install -r requirements.txt
     ```

6. (Opcional) Configure variáveis de ambiente
   - O arquivo `config.py` define `HOST` e `PORT`. Por padrão, o servidor roda em `localhost:5050`. 
   - Se desejar outra porta ou host, edite `config.py` ou defina as variáveis no terminal:
     ```bash
     set HOST=0.0.0.0     # Windows
     set PORT=8000        # Windows
     export HOST=0.0.0.0  # Linux/macOS
     export PORT=8000     # Linux/macOS
     ```

7. Execute o servidor Flask
   - Com o ambiente virtual ativo, rode o arquivo principal `add.py`:
     ```bash
     python add.py
     ```
   - O console mostrará uma mensagem como `Starting Flask server on http://127.0.0.1:5050`. Deixe essa janela aberta.

8. Abra a interface web no navegador
   - Em um navegador, abra o arquivo `index.html` da pasta `frontend/` (por exemplo, `file:///C:/.../frontend/index.html`), ou configure um servidor estático para servi-lo.
   - Preencha a “Chave do Consumidor” e “Segredo do Consumidor”, clique em “Conectar” para obter o token e use os filtros de busca.

9. Encerrar o servidor
   - Quando não precisar mais do backend, volte ao terminal onde ele está rodando e pressione `Ctrl+C` para interromper o processo.
