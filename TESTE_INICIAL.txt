1. VERIFICAR PRÉ-REQUISITOS
   1.1 Verificar Python 3.11+ instalado
       - Abra CMD ou PowerShell e digite:
         python --version
       - Deve retornar: Python 3.11.x
   1.2 Verificar Pip instalado
       - Comando: pip --version
   1.3 Verificar dependências do projeto
       - No CMD ou PowerShell, dentro da pasta do projeto:
         pip install -r backend\requirements.txt
       - Isso instala Flask, requests e flask-cors

2. CONFIGURAR A API
   2.1 Abra o arquivo backend\config.py
   2.2 Verifique se as chaves da API estão corretas:
       CONSUMER_KEY = "1rQmU8QENUAmUHWxnvupVUeNwPwa"
       CONSUMER_SECRET = "Zd9fp7G41pDE8Jn6A4FsfnbKmTEa"
       ACCESS_TOKEN = "279b6844-5c57-3c02-8745-8bcacddc1748"
   2.3 Defina o endpoint para testes iniciais:
       API_URL = API_SANDBOX

3. INICIAR O BACKEND (Servidor Flask)
   3.1 Abra CMD ou PowerShell na pasta do projeto:
       cd C:\caminho\para\busca_cad_escolas_sp\backend
   3.2 Ative o virtualenv (se estiver usando):
       CMD: venv\Scripts\activate
       PowerShell: .\venv\Scripts\Activate.ps1
   3.3 Inicie o servidor Flask:
       python app.py
   3.4 Verifique se aparece:
       * Running on http://127.0.0.1:5000/
   3.5 Obs.: Se a porta 5000 estiver ocupada, feche outro servidor ou altere a porta no app.py

4. ABRIR O FRONTEND
   4.1 Abra o arquivo frontend\index.html no navegador
   4.2 Você verá a interface com:
       - Campo de busca
       - Botão "Buscar"
       - Botão "Limpar"

5. TESTAR A BUSCA INICIAL
   5.1 Digite o nome de uma escola, por exemplo: "Escola Municipal ABC"
   5.2 Clique em "Buscar"
   5.3 O resultado aparecerá abaixo do botão em formato JSON
   5.4 Clique "Limpar" para resetar o campo e o resultado

6. VERIFICAR POSSÍVEIS ERROS
   6.1 Se der erro de conexão:
       - Confirme que o backend está rodando
       - Verifique se firewall do Windows permite a porta 5000
   6.2 Para usar Produção:
       - Altere API_URL = API_PRODUCAO em backend\config.py
       - Reinicie o servidor Flask

7. DICAS FINAIS
   - Sempre iniciar o backend antes do frontend
   - Sandbox é seguro para testes
   - Proteja suas chaves da API
   - Para resultados mais legíveis, futuramente implementar renderização em tabela
